## Stack utilizada

**Back-end:** Python, FastAPI, Uvicorn

**Cloud:** AWS, Terraform


## Rodando localmente

Clone o projeto

```bash
  git clone https://github.com/alunounifae/projeto-devops
```

Entre no diretório do projeto

```bash
  cd task-api
```

Instale as dependências

```bash
  pip install -r requirements.txt
```

Inicie o servidor

```bash
  python main.py
```
